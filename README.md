# ✨ Bem-vindo ao Editor Hyperlink

Para começar, acesse o editor:  
👉 [Editor Hyperlink](https://srshadowy.github.io/hyperlink/editor.html)

---

## 📂 Já tem um `data.json`?
Se você já possui um arquivo `data.json` do seu Hyperlink, basta carregá-lo no editor e começar a editar.

---

## 🚀 Primeira vez por aqui?
Sem problemas! Vamos juntos nessa jornada:

1. Crie seu `data.json` diretamente no editor: 👉 [Abrir Editor](https://srshadowy.github.io/hyperlink/editor.html)  

2. Na aba *JSON final*, use o botão **Download pacote** para baixar os arquivos necessários e publicar seu link.  

3. Personalize seu projeto adicionando imagens na pasta `/resources/icons`.  

4. Gere o JSON no editor e copie-o com o botão **COPIAR**.  

5. Abra seu editor de texto preferido, cole o conteúdo e salve como `data.json` dentro da pasta `/resources`.  

---

## 🧹 Quer começar do zero?
Use o botão **LIMPAR EDITOR** na aba *JSON final* para apagar tudo e recomeçar.

---

💡 **Dica:** mantenha sempre uma cópia do seu `data.json` salva em algum lugar seguro. Assim você pode carregar novamente quando quiser.
